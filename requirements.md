## Packages
papaparse | For parsing and exporting CSV data smoothly in the browser
@types/papaparse | TypeScript definitions for papaparse
xlsx | For parsing Excel (XLSX) files in the browser
recharts | For building the professional charts in the dashboard
clsx | For conditional tailwind class merging
tailwind-merge | For merging tailwind classes safely

## Notes
- All data processing occurs locally in the browser to ensure zero latency and privacy.
- No backend API endpoints are used for data mutation or storage.
- Using `papaparse` for CSV and `xlsx` for Excel parsing directly via FileReader.
